# Precious Home Help

Personal, owner-operated house cleaning marketing site for Bukola Precious Agbenike in Chicago.

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS v4 + shadcn/ui
- Fonts: Fraunces (serif display) + Inter (sans body)
- Email: Resend (quote form → Precious's inbox)
- Forms: react-hook-form + zod
- Blog: MDX via next-mdx-remote
- Hosting: Vercel

## Key Rules
- **No "our team" / "our cleaners" / "we'll send someone"** — solo operator voice only
- **Auto-update RELEASE_NOTES.md and USER_GUIDE.md** after every change
- **TypeScript strict mode** — no `any` unless justified
- **Mobile-first responsive** — 320px+ is first-class
- **Accessibility** — all images need alt, form inputs need labels, WCAG AA contrast
- **Push after changes** — Scott tests via Vercel preview deploys
